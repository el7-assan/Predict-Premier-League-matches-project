# Predict-Premier-League-matches-project
Intelligent platform for predicting Premier League match results using artificial intelligence.
Key Features:

Winner prediction (Home Win – Draw – Away Win) with displayed probabilities.
Interactive interface: Select the two teams → instantly display their logos + stadium image + key team statistics.
Powerful backend built with FastAPI for handling predictions.
Simple and fast frontend (HTML + CSS + JavaScript).

Technologies Used:

Comprehensive data analysis and cleaning (fixtures, player stats, salaries, team stats…).
Merging multiple tables + strong feature engineering.
Machine learning models .
